These are some of the shortcode that needs to be placed in the pages or post to actually see the result.


[user_biodata_register_login] to display the register and login section.
[user_biodata_add_edit] to allow logged-in users to add or edit their biodata.
[user_biodata_pending] to show the pending biodata section for the admin.
[user_biodata_list] to display the list of approved biodata candidates.
